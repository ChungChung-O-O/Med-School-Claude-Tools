### h. Image Occlusion — Proven, Not Yet Automated

Verified that native Image Occlusion cards can be built programmatically via AnkiConnect without the Enhanced IO add-on:

1. `storeMediaFile` — upload PNG to Anki media folder
2. `addNote` — model: `"Image Occlusion"`, Occlusion field uses the format:
   `{{cN::image-occlusion:rect:left=<x>:top=<y>:width=<w>:height=<h>}}`
   where coordinates are 0–1 fractions of image dimensions

**Coordinate source for labeled slides:** PyMuPDF `page.get_text("words")` gives bounding boxes for text labels with sub-pixel precision. For flat images without text layers, use Claude vision to identify label positions.

**Status:** Not yet wired into the pipeline. Wire it up when the user sends the first real labeled anatomy slides.
